Nastavenie MCP ▶

🟠 Claude.ai Recommended
  1. Go to claude.ai → Settings → Integrations
  2. Click "Add custom integration"
  3. Enter URL: https://oncofiles.com/mcp
  4. Click "Connect" → sign in with Google
  5. Select patient when prompted
Features: 78 tools, OAuth login, auto-patient selection
Best for: Full medical document analysis, lab trends, treatment planning
🟢 ChatGPT
  1. Go to chatgpt.com → Settings → Connected Apps
  2. Click "Add MCP Server"
  3. Enter URL: https://oncofiles.com/mcp
  4. Authenticate with Google OAuth
  5. Select patient in the chat
Features: 78 tools, OAuth login
Best for: Quick document queries, second opinion analysis
🔵 Bearer Token (API)
  1. Generate token in dashboard (above)
  2. Configure in your MCP client:
{
  "mcpServers": {
    "oncofiles": {
      "url": "https://oncofiles.com/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_TOKEN"
      }
    }
  }
}
Features: 78 tools, token-scoped to one patient
Best for: Claude Code, Cursor, Windsurf, custom integrations
Patient Isolation: Each connection is scoped to one patient. AI Client A never sees Patient B's documents.

GDrive Manifest ▶

_manifest.json is exported every 5 minutes to your GDrive sync folder root. Contents:

  • documents — all files with metadata, categories, summaries, tags
  • treatment_events — chemo cycles, labs, consults, surgeries
  • conversation_entries — session notes, decisions, observations
  • research_entries — PubMed articles, clinical trials, protocols
  • agent_state — internal state (costs, scan timestamps, etc.)

Use it for disaster recovery, auditing, or feeding data to other tools.
